- is [[Periodic Signal]]
- #notation #formula

	- $x(t) = Asin(w_{0}t + \phi)$
		- $A$   : Amplitude
		-  $w_{0}$ : speed $(rad / s)$
		- $\phi$   : starting phase
	- We know that $w_{0} = 2\pi f$
	
	- we can calculate period $(T)$ using the #formula
	  $T = \frac{2\pi}{|w_{0}|} = \frac{2\pi}{2\pi f} = \frac{1}{f}$   

	- we can also calculate $w$ using parameter $T$ using the #formula
	  $w = \frac{2 \pi}{T}$
